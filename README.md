# Rasterrun

Rasterrun is a 3D action game made in Godot, in which you have to navigate various obstacles, unable to ever stop or go back. The game is playable on itch.io

## Building

Rasterrun was entirely developped using Godot 4.5.1 stable, and can/should be modded using that exact version. It uses the Compatibility render.
